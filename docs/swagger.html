OpenAPI 3 + Swagger UI

A documentação da API é gerada automaticamente através do SpringDoc OpenAPI seguindo o padrão de interfaces dedicadas para documentação. 

controller/
├── TranscriptionController
└── documentation/
    └── TranscriptionControllerDocumentation

Essa abordagem mantém os controllers focados na adaptação HTTP e concentra anotações OpenAPI em interfaces específicas.

  • /swagger-ui.html
  • /v3/api-docs
