# V3.7.0 — Migration Flyway Inicial

## Visão Geral

Esta etapa introduz a primeira migration versionada da aplicação utilizando Flyway.

Foi criada a estrutura inicial da tabela `transcriptions` juntamente com índices e restrições voltadas para futura deduplicação por `audioHash`.

A migration estabelece a base persistente do bounded context de transcrição mantendo evolução segura e rastreável do schema.

---

# O que foi implementado

## Tabela transcriptions

Foi criada a tabela principal responsável pelo armazenamento das transcrições persistidas.

### Estrutura inicial

```sql
CREATE TABLE transcriptions (
    id UUID PRIMARY KEY,
    audio_hash VARCHAR(64) UNIQUE,
    transcription TEXT
);
```

---

## Campos da tabela

| Campo | Objetivo |
|---|---|
| id | Identificador UUID |
| audio_hash | Deduplicação |
| transcription | Conteúdo transcrito |
| created_at | Timestamp persistido |

---

## Índice único

Foi criado índice único para `audio_hash`.

### Benefícios

| Benefício | Impacto |
|---|---|
| Busca rápida | Melhor performance |
| Consistência | Evita duplicações |
| Reuso inteligente | Deduplicação futura |
| Base para cache | Chave estável |

---

# Decisão de arquitetura

## Por que Flyway ao invés de geração automática?

Migrations explícitas garantem:

- Rastreabilidade
- Segurança
- Evolução controlada
- Compatibilidade futura

---

## Por que índice único no audio_hash?

O hash representa identidade lógica do conteúdo do áudio.

A unicidade garante consistência na reutilização futura.

---

## Trade-offs adotados

| Decisão | Benefício | Trade-off |
|---|---|---|
| Flyway | Evolução segura | Necessidade de migrations |
| UUID | Segurança | Índices maiores |
| Unique hash | Consistência | Restrições rígidas |
| SQL explícito | Controle total | Mais manutenção |

# Troubleshooting

## Erro: Flyway checksum mismatch

### Causa

Migration alterada após execução.

### Solução

Nunca editar migration já aplicada.

Criar nova migration incremental.

---

## Erro: relation "transcriptions" does not exist

### Causa

Migration não executada.

### Solução

Validar startup do Flyway.
