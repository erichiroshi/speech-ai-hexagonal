# Fase 2 Item 4 — Propriedades do SonarQube

## O que foi implementado

- Variáveis de ambiente do Sonar
- Integração dos relatórios XML
- Parametrização segura

## Arquivos criados/alterados

- build.gradle
- .env.example
- README.md
- docs/security.html

## Decisão de arquitetura

As credenciais foram externalizadas para evitar exposição no Git.

Trade-off:
- maior setup local
- maior segurança operacional

## Como testar

```bash
export SONAR_TOKEN=local
./gradlew sonar
```

## Troubleshooting
Token inválido

Verificar:
```bash
SONAR_TOKEN
SONAR_HOST_URL
```
